Example Notebooks¶

CmapExtractor Demo¶

notebook html

Walkthough of yoink.cmap_app.CmapExtractor showing the extraction of a scalar field from a colormapped image. Shows integrated cropping, colorbar selection, and colormap inversion. Also, manual specification of x-axis, y-axis and colorbar limits.

Manual Colormap Extraction¶

notebook html

Very explicit usage of yoink widgets to extract the data from a colormapped image. Indended to serve as a template for more complicated extractions.

Manual Line Extraction¶

notebook html

Very explicit usage of yoink widgets to extract the data from a line plot. Indended to serve as a template for more complicated extractions.
